# web-design

brand-specific skill for live interactive web craft — Next.js landing pages, HubSpot landing pages, and production web UI. See `SKILL.md` for the entry point.

## Attribution

The architecture of this skill (split reference files by craft domain + named workflows like audit/critique/polish/typeset/layout) is inspired by **[impeccable](https://github.com/pbakaus/impeccable)** by Paul Bakaus, MIT-licensed. Impeccable was built specifically to fight the generic AI aesthetic that the stock `frontend-design` skill tends to produce — an excellent prior for any brand-heavy product.

This skill borrows the *structure* only. All content is brand-specific and pulled from `brain/brand-guide/brand-guide.md` and `[your-site-repo-path]/src/`.

## Boundary

- **`web-design`** — interactive web craft (this skill)
- **`brand-design`** — static creative assets (ads, carousels, slide graphics, whitepaper PDFs)
- **`hubspot-email`** — HubSpot email HTML templates
- **`hubspot-landing-page`** — HubSpot template mechanics, HubL, Design Manager
- **`hubspot-cta`** — HubSpot CTA button modules
- **`image-gen`** — AI-generated illustrations via Gemini
- **`frontend-design` (Anthropic plugin)** — generic frontend work; coexists, `web-design` wins for anything on-brand
