## General Rules

- Always design at 2x for retina displays when possible
- Export as PNG for graphics with text, JPEG for photos
- SVG preferred for logos and icons
- Keep file sizes reasonable — compress PNGs without quality loss
- Test readability at actual display size, especially on mobile
