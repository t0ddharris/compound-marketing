# Output Templates

Templates for every deliverable the product-marketing skill produces. Use these as structural guides — adapt depth and detail to the specific task.

---

## 1. Positioning Statement

### Dunford-Method Positioning Canvas

```
COMPETITIVE ALTERNATIVES
What would the buyer do if we didn't exist?
→ [List 3-5 alternatives including status quo, DIY, direct competitors]

UNIQUE ATTRIBUTES
What do we have that alternatives lack?
→ [List attributes — must be provable, not aspirational]

VALUE (mapped from attributes)
→ Attribute: [X] → Value: [What this enables for the customer]
→ Attribute: [Y] → Value: [What this enables for the customer]
→ Attribute: [Z] → Value: [What this enables for the customer]

TARGET CUSTOMER CHARACTERISTICS
Who cares most about this value?
→ [Describe characteristics, not demographics]

MARKET CATEGORY
What context makes our value obvious?
→ [Category frame]
```

### Positioning Statement (External Format)

```
For [target audience with specific characteristics]
Who [situation that creates urgency]
[Product] is a [market category]
That [primary value — what they get]
Unlike [competitive alternatives]
[Product] [key differentiator — what only we do]
```

```

[One sentence — the single most important thing]

Pillar 1: [Theme Name]
  Headline: [One sentence]
  Proof Points:
    - [Specific, verifiable claim]
    - [Specific, verifiable claim]
    - [Specific, verifiable claim]
  Use When: [Audience + context]
  Customer Quote: [If available]

Pillar 2: [Theme Name]
  ...

Pillar 3: [Theme Name]
  ...

Pillar 4: [Theme Name]
  ...

ELEVATOR PITCHES
  25 words: [Core identity and value]
  50 words: [+ capability and proof]
  75 words: [+ problem context and competitive frame]

DESCRIPTION BLOCKS
  Short (1 sentence): [Core value prop]
  Medium (2-3 sentences): [+ how and proof]
  Long (paragraph): [+ problem context, customers, full positioning]
```

## 7. Sales Narrative

```
SALES NARRATIVE: [Context — e.g., "First discovery call" or "Executive presentation"]

TARGET AUDIENCE: [Role + buying stage]

OPENING (Problem framing — 60 seconds)
[Set up the problem in the buyer's language. No product yet.]

INSIGHT (The "aha" — 30 seconds)
[The non-obvious truth about why their current approach fails]

SOLUTION FRAMING (How we address it — 90 seconds)
[Connect insight to your approach. Still outcome-focused, not feature-focused.]

PROOF (Evidence — 60 seconds)
[Customer example, metric, or demonstration that validates the claim]

DIFFERENTIATION (Why us — 60 seconds)
[What we do that alternatives cannot. Specific and verifiable.]

CALL TO ACTION
[Next step appropriate to the buying stage]

OBJECTION RESPONSES
- If they say: "[Objection]" → Response: "[Talk track]"
- If they say: "[Objection]" → Response: "[Talk track]"
```
